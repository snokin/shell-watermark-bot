## 这是个啥？有啥功能？
- Telegram 水印机器人，给视频打水印的
- 不同用户可以设置自己的水印文件
- 单文件 Shell 脚本
- 开频道偷视频一绝

## 怎么使用？
- 依赖：需要有 jq ，没有安装一个 `apt isntall jq`
- 依赖：需要有 ffmpeg ，没有安装一个 `apt install ffmpeg`
- @botfather 处申请一个 bot
- 将 bot token 填到文件内 `<bot_token>` 处
- 把机器人运行的主目录设置到文件内 `/your/path/to/dir` 处
- 主目录内准备个 .png 的水印文件（用户不设置水印就会用这个）
- `sh ./wmbot.sh`
